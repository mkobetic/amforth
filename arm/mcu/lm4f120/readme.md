# TI Stellaris LM4f120 LaunchPad



# MCU

LM4F120H5QR 32-bit ARM Cortex M4F (renamed to Tiva TM4C1233H6PM)
* 256 kB flash memory
* 32 kB SRAM: $20000000 .. $20008000
* 80 MHz

# Peripherals

## GPIO
* PF4 SW1
* PF0 SW2
* PF1 LED (red)
* PF2 LED (blue)
* PF3 LED (green)
